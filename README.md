# VOTESPHERE
VoteSphere is a next-generation web-based voting platform built using HTML, CSS, and JavaScript. It provides a complete solution for creating and managing elections with an intuitive and visually rich user interface.

The application supports multiple voting methods including plurality, ranked choice, approval, and score voting, making it flexible for different use cases such as student elections, surveys, and organizational decision-making.

Key features include:

🗳️ Multi-election support with dynamic switching
🔒 Secure and anonymous voting experience
📊 Real-time results with charts, leaderboards, and analytics
⚙️ Admin panel to create elections, manage candidates, and control voting status
🔗 Shareable voting links and QR code generation
👥 Voter management system with tracking and simulation options
🎨 Modern UI with themes, animations, and responsive design

This project demonstrates strong front-end development skills, UI/UX design, and the implementation of complex interactive features without relying on external frameworks.
